First run the user.py file to execute the project,it will ask to choose the options 1 if you are a super user for adding questions
and 2 for user to take the quiz.

Note: Before connecting to Database run this command:- pip install tiny-orm
Input format for question is: ex:"When does India celebrate Independence Day?"
Input format for choices is: ex: "A. 15th August",
		"B. 2nd October",
		"C. 26th January",
		"D. 4th July"
Input format for answer is: ex: a or b or c or d
